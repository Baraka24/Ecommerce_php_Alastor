E-Commerce Website (Alastor)
Welcome to our E-Commerce Website Alastor!

This PHP-based website provides a platform for users to buy and sell products online. Below are the key features of our website:

Features
User Authentication
Login: Users can securely log in to their accounts.
Registration: New users can create accounts by providing necessary information.
Product Management
Create Article: Sellers can add new articles/products to the website.
Delete Article: Sellers can remove articles/products from their inventory.
Shopping
View Products: Users can browse through a wide range of products available on the website.
Add to Cart: Users can add products to their shopping cart for purchase.
Checkout: Users can proceed to checkout to complete their purchase.
Orders
Place Order: Users can place orders for the selected products.
View Order History: Users can view their order history to track previous purchases.
Getting Started
To run the website locally, follow these steps:

Clone this repository to your local machine:

bash
Copier le code
git clone https://github.com/softadastra/Ecommerce_php_Alastor.git
Import the database schema provided in the database.sql file to set up the database.

Configure the database connection in the config.php file.

Start a local PHP server or use a web server like Apache to host the website.

Access the website in your web browser and start exploring the features!
