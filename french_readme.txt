Site E-Commerce (Alastor)
Bienvenue sur notre site E-Commerce Alastor

Ce site web basé sur PHP offre une plateforme permettant aux utilisateurs d'acheter et de vendre des produits en ligne. Voici les principales fonctionnalités de notre site :

Fonctionnalités
Authentification des Utilisateurs
Connexion : Les utilisateurs peuvent se connecter de manière sécurisée à leur compte.
Inscription : Les nouveaux utilisateurs peuvent créer des comptes en fournissant les informations nécessaires.
Gestion des Produits
Créer un Article : Les vendeurs peuvent ajouter de nouveaux articles/produits sur le site.
Supprimer un Article : Les vendeurs peuvent supprimer des articles/produits de leur inventaire.
Shopping
Visualiser les Produits : Les utilisateurs peuvent parcourir une large gamme de produits disponibles sur le site.
Ajouter au Panier : Les utilisateurs peuvent ajouter des produits à leur panier d'achat pour les acheter.
Passer à la Caisse : Les utilisateurs peuvent passer à la caisse pour finaliser leur achat.
Commandes
Passer une Commande : Les utilisateurs peuvent passer des commandes pour les produits sélectionnés.
Voir l'Historique des Commandes : Les utilisateurs peuvent consulter leur historique de commandes pour suivre leurs achats précédents.
Pour Commencer
Pour exécuter le site localement, suivez ces étapes :

Clonez ce dépôt sur votre machine locale :

bash
Copier le code
git clone https://github.com/softadastra/Ecommerce_php_Alastor.git
Importez le schéma de la base de données fourni dans le fichier database.sql pour configurer la base de données.

Configurez la connexion à la base de données dans le fichier config.php.

Démarrez un serveur PHP local ou utilisez un serveur web comme Apache pour héberger le site.

Accédez au site web dans votre navigateur web et commencez à explorer les fonctionnalités !
